# identify

**Module:** `@arcgis/core/rest/identify`

## Import

```javascript
import * as identify from "@arcgis/core/rest/identify.js";
```

```javascript
// CDN
const identify = await $arcgis.import("@arcgis/core/rest/identify.js");
```

**Since:** 4.19

## Overview

Performs an identify operation on the layers of a map service exposed by the ArcGIS Server REST API. Use IdentifyParameters to set the parameters for the identify operation and IdentifyResult to work with the results. Known Limitations The identify operation is currently not supported if attempting to be used in a 3D SceneView
