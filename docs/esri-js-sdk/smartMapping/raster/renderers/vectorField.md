## Overview

This object contains helper methods for generating a VectorFieldRenderer for a Vector-UV or Vector-MagDir ImageryLayer or ImageryTileLayer. The createRenderer method in this module generates a renderer that may be applied directly to the input layer. Known Limitations VectorFieldRenderer is only supported with ImageryTileLayer and ImageryLayer where the source type is Vector-UV or Vector-MagDir. VectorFieldRenderer is only supported in 2D MapView.
